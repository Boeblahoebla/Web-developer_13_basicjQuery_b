jQuery common methods

This is a demonstration of some common jQuery methods

This project is used as a demo of the most used jQuery methods like

  • .val()
  • .text()
  • .attr()
  • .html()
  • .addClass()
  • .removeClass()
  • .toggleClass()

Each topic is clarified in its own section having an identified div with some content illustrating its functionality
